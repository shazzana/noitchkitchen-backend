# No-itch Kitchen

## About the App

This purpose of the site is to provide a tool for people with histamine intolerance to search food ingredients and get information on how the ingredients rate in terms of histamine compatibility and the mechanisms which the ingredients use to produce histamine. 

- Built as part of a project for a software engineering course.
- MERN stack app

### Site can be accessed at the following url: https://noitchkitchen.vercel.app/

- This repo consist of the backend of the site
- The frontend uses HTML, CSS, Javascript, ReactJS. It can be accessed here: https://github.com/shazzana/noitchkitchen
- Both repos must be running and for the site to work locally
- Environment variables for MongoDB, Firebase and port must be present in an .env file for the app to be deployed successfully

## Tech Stack/Libraries for backend

- Javascript
- Node.js
- MongoDB
- Express
- Bootstrap

## API used
